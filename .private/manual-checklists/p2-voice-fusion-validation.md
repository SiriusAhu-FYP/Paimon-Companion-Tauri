# P2.2 语音融合检查单

填写规则：

- 每项都填写 `是否成功`
- 若结果不是“成功”，再填写 `现象或输出`
- 可选值：`成功` / `部分成功` / `失败` / `阻塞`

前置说明：

- 若只运行 `pnpm tauri dev`，会自动准备本地 sherpa 资源。
- 若单独运行 `cargo check`，请先执行一次 `pnpm setup:local-asr`。
